# ESOL Community Resource Center

A responsive, accessible static website prototype created by Natalie Gill.

## What it includes
- Searchable and filterable resource library
- Family and educator resource sections
- Interactive lesson differentiation builder
- Support-request form that stores demo submissions locally in the browser
- Mobile-responsive design
- No backend or API required

## How to run
Open `index.html` in a browser.

## How to publish free
You can upload the folder to:
- GitHub Pages
- Netlify
- Cloudflare Pages

## Important note
This is a prototype. Community referrals and school-specific guidance should be verified before public use.
