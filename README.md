# Bridge ESOL

Bridge ESOL is a multilingual educational resource website created by Natalie Gill.

## Included
- Responsive homepage inspired by the Bridge ESOL design mockup
- Custom community illustration
- English, Spanish, Haitian Creole, and Portuguese language selector
- Family, educator, and community resource sections
- Mission and contact sections
- Mobile navigation
- GitHub/Netlify-ready static files

## Upload to GitHub
Replace the current repository files with:
- index.html
- styles.css
- script.js
- assets/bridge-esol-community.png

Then commit the changes. Netlify will deploy the update automatically.
